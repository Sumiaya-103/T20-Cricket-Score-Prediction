
## **T-20 Cricket Match Score Prediction (Machine Learning Project)**

This project predicts the final score of a T-20 cricket match using machine learning regression techniques. The model analyzes match progress features—such as current run rate, wickets fallen, and last overs performance—to estimate the final innings score with high accuracy.

---

### **Project Overview**

This project applies supervised machine learning to analyze cricket match dynamics and forecast the total score. The workflow includes:

* Data cleaning and preprocessing
* Feature engineering
* Model training and evaluation
* Performance comparison between ML models
* Visualization and result interpretation


### **Machine Learning Models Used**

* **Decision Tree Regressor**
* **Random Forest Regressor** (Best performer)

**Best Accuracy Achieved:**
 **Random Forest — 92% test accuracy**
